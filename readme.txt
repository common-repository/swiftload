Add a stylish preloader to your website with ease, featuring customizable styling and animations. This feature is compatible with all major browsers, adapts to any theme, and is optimized to work perfectly on any device.

== Description ==

Introduce a preloader while your website page loads all components, providing a smoother experience for your visitors.

##EASY TO CUSTOMIZE

SWIFTLOAD is integrated into WordPress settings, allowing you to adjust all settings intuitively and see results immediately.

##WORKS ANYWHERE

Swiftload is compatible with all major browsers, offering versatility to be used on any theme and device.

##SWIFTLOAD FEATURES

SWIFTLOAD is easy to configure and customize, you can choose between one of the 4 icons, add a progression bar or a percentage that comes by default.

== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.

== Frequently Asked Questions ==

= How can I change the settings of the preloader =

Navigate to Settings > SWIFTLOAD WP. Here you find the panel with all the SWIFTLOAD WP settings.

= How to uninstall the plugin? =

Simply deactivate and delete the plugin.
